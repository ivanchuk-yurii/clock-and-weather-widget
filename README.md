# Clock and Weather Widget

A lightweight Angular widget that displays the current time and weather information.
[OpenWeather](https://docs.openweather.co.uk/current) API is used for weather data.

## Requirements

- Node.js >= 18 (20 LTS recommended)
- npm >= 11
- Angular CLI >= 21

## Development

```bash
$ npm i
$ npm start
```

## Formatting

```bash
$ npm run format
```

## Build

```bash
$ npm run build
```
